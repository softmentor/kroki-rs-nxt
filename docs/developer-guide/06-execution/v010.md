- Added end-user runtime dependency bootstrap script (`scripts/install-runtime-deps.sh`) and updated docs.

- Added host-only verification command `dwf setup:host-deps`.
- Added install mode for host dependency remediation:

  - optional Chromium install toggle via `KROKI_INSTALL_CHROMIUM=1`
- Updated developer and user docs to reflect the new `dwf` host dependency workflow and command matrix.

### 2026-03-03 (Native Browser/Font Port Progress)

  - `browser/manager.rs` manager bootstrap
  - `browser/native.rs` `headless_chrome` (CDP) backend
- Wired Mermaid provider to prefer the native browser manager in `native-browser` builds, with `mmdc` fallback retained for resilience and compatibility.
- Implemented browser pool and lifecycle controls:
  - pool size via `KROKI_BROWSER_POOL_SIZE`
  - request-count TTL recycle via `KROKI_BROWSER_CONTEXT_TTL_REQUESTS`
  - adaptive recycle via `KROKI_BROWSER_ADAPTIVE_RECYCLE_FAILURES`
- Implemented font management for browser rendering:
  - cache-backed font fetch/copy (`utils/font_manager.rs`)
