# Developer Guide

Welcome to the kroki-rs-nxt developer guide. This guide is organized as a practical path from setup, to architecture, to day-to-day execution and delivery.

## How to Use This Guide

If you are new to the repository, start with **Quickstart** and **Repository Structure**.
If you are actively building features, use **Workflow**, **Build & Release**, and the **Execution** tracker.
If you are planning architecture changes, review **Architecture** and the **ADRs** first.
